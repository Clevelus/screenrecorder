Screen Recorder
Records user specified part of Windows desktop screen as an AVI file. The project is released under the MIT License and version 1.2-beta is available for download.
Features:
    User-specified capture region in three ways:
        Full Screen
        Fixed part of the screen
        Window region.
    User-specified video compressor and fps (frames per second)
    Show/Hide mouse cursor in the recorded video.
    Customizable hot keys for the recording operations
    Easy to use and friendly user interface

System Requirements:
    Windows XP, Vista, 7, 8, 8.1 or 10
    Microsoft .NET Framework 2.0 SP2

Setup files, screenshots and more are available in:
https://sourceforge.net/projects/screenrecorder-x
